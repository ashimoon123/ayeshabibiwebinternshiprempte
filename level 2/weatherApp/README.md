# NovaStack Weather App (Enhanced)

A modern, visually stunning weather application featuring glassmorphic design with dynamic animated backgrounds and real-time weather information.

## Features

- **Modern UI Design**: Glassmorphic card design with smooth animations and transitions
- **Dynamic Background**: Animated floating orbs that respond to weather conditions
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Real-time Weather Data**: Integration with weather APIs for current conditions
- **Smooth Animations**: Polished animations and hover effects for enhanced user experience
- **Weather Icons**: Font Awesome 6 icons for weather condition visualization

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with glassmorphism, backdrop filters, and animations
- **Google Fonts**: Inter font family for typography
- **Font Awesome 6**: Weather and utility icons
- **Responsive Design**: Mobile-first approach with flexible layouts

## Design Highlights

- **Color Scheme**: Dark theme with blue and teal accents
- **Backdrop Filters**: Blur and saturation effects for modern look
- **Animations**: Floating orbs and smooth transitions
- **Typography**: Clean Inter font with varied weights for hierarchy
- **Shadows & Depth**: Layered box shadows for dimensional appearance

## Getting Started

1. Open `index.html` in your web browser
2. Allow location/weather data access if prompted
3. View current weather information and forecasts

## Browser Support

Works best in modern browsers that support:
- CSS Backdrop Filters
- CSS Grid and Flexbox
- CSS Animations
- Fetch API
