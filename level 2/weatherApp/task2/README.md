# Pakistan Weather - Live Forecast

A comprehensive weather forecasting application focused on Pakistan with live weather updates, particle animations, and a modern responsive interface.

## Features

- **Live Weather Data**: Real-time weather information for Pakistani cities
- **Animated Background**: Particle animation system with floating elements
- **Responsive Design**: Fully responsive layout optimized for all screen sizes
- **Dynamic Background Gradients**: Color-changing background based on weather conditions
- **Modern UI**: Clean, professional interface with glassmorphic elements
- **Weather Icons**: Font Awesome 6 icons for intuitive weather visualization

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with animations, gradients, and responsive design
- **Google Fonts**: Inter font family for consistent typography
- **Font Awesome 6.4**: Comprehensive weather and utility icons
- **JavaScript**: (If included) For interactive functionality and API integration

## Design Features

- **Color Scheme**: Dark blue theme (#0a1628) with accent colors
- **Particle System**: 12 animated particles with varying speeds and delays
- **Header Section**: Glassmorphic header with blur and transparency effects
- **Responsive Grid**: Mobile-first design with flexible layouts
- **Animations**: Smooth floating particle effects and transitions

## UI Components

- **Background Animation**: Radial gradient with animated particles
- **Header**: Sticky navigation with search and settings
- **Weather Container**: Grid layout for displaying weather cards
- **Particle Effects**: 12 independent floating particles with unique animations

## Getting Started

1. Navigate to the `task2` folder
2. Open `index.html` in your web browser
3. Grant location permissions for accurate weather data
4. Browse weather forecasts for Pakistani cities
